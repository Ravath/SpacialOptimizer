A tool for optimizing the positioning of different elements.

Currently using Godot 4.2

Default shortcuts (QWERTY configuration) :
- w/s : Attraction/repulsion up/down
- e : Create node
- r : Remoce node
- d : Duplicate node
- s : Split node
- f : fix node
- t : Generate random nodes
- y : Reset simulation
- \<space> : pause simulation
